## 📜 阅前须知

此仓库用于收集、索引 **基于 XRK-AGT 开发的所有项目与工具**，方便统一检索与对照学习。

本仓库仅作为 **索引 / 导航** 使用，对被索引内容本身（代码质量、安全性、维护状态等）**不做全面验证与担保**。在使用任何第三方代码或脚本前，请务必自行评估风险。

本仓库默认以 **XRK-AGT 最新版本及其生态** 为主，旧版本/兼容性请根据实际情况自行判断。

---

## 📤 如何提交索引

如果你有基于 XRK-AGT 开发的项目、工具或教程，欢迎发起 Pull Request：

- **新增项目/工具索引**：补充到对应分类表格中（按从上到下，新到旧排序）  
- **修正信息**：如作者、仓库地址、说明等有变更，可直接修改对应行  
- **补充文档/教程**：可在教程类资源中新增条目

（可根据项目需要在仓库中补充 `CONTRIBUTING.md`，并在此处放上链接）
